## 📦 Release 说明

### 版本命名规则

- **`vX.Y.Z`**：正式应用版本，需手动下载并安装。
- **`@YYYY.MM.DD`**：续命脚本快照，每次页面刷新时自动更新，**无需手动操作**。

### 发布产物类型

| 文件后缀         | 类型说明             | 推荐场景               |
|------------------|----------------------|------------------------|
| `-setup.exe`     | 安装包版本           | 首次安装或长期使用     |
| `-portable.exe`  | 单文件便携版         | 临时使用或免安装需求   |

> 💡 如无特殊需求，**建议使用 `-setup.exe` 后缀的安装包版本**。
