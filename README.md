# 关于这个仓库

本仓库用于发布天翼云电脑续命工具的 [Release 版本](https://github.com/mzl980425/CTYunPCKeepAlive/releases) 以及使用 [Github Issues](https://github.com/mzl980425/CTYunPCKeepAlive/issues) 收集和跟进用户反馈。

天翼云电脑续命工具并非开源软件，这个仓库并 **不包含** 天翼云电脑续命工具的源代码。

## Release 说明

### 版本号规则

- 以 `v` 开头的版本号是应用版本，安装或更新都需要手动下载并安装
- 以 `@` 开头的日期版本号是续命脚本，每次页面刷新后会自动执行更新，无需手动操作

### 产物文件

> 如无特殊诉求，建议使用安装包版本

- 以 `-portable.exe` 结尾的文件是单文件便携包版本
- 以 `-setup.exe` 结尾的文件是安装包版本
